# Daily-land-use-regression
# 使用说明

在python3.7下进行编写，使用前在项目根目录执行以下命令以安装依赖：

```bash
pip install -r ./requirements.txt
```

安装依赖后于项目根目录执行以下命令以运行脚本：

```bash
python script.py
```
